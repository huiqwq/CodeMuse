# 版本文档索引

每个 CodeMuse 版本必须新增独立文档，文件名为 `v<版本号>.md`。

- [v0.1.0](v0.1.0.md)：CLI 与模型调用基线。
- [v0.2.0](v0.2.0.md)：只读代码库分析 Agent。
- [v0.3.0](v0.3.0.md)：任务规划与智能上下文管理。
- [v0.4.0](v0.4.0.md)：安全局部修改、Diff 授权与撤销。
- [v0.5.0](v0.5.0.md)：受控 npm scripts、构建和测试验证。
- [v0.6.0](v0.6.0.md)：失败诊断、受控自动修复和停止策略。
- [v0.7.0](v0.7.0.md)：本地会话历史、工作区检查点和恢复。
- [v0.8.0](v0.8.0.md)：完整文件操作、只读 Git 审查和变更归属。
- [v0.9.0](v0.9.0.md)：本机多模型配置、切换、连接测试、重试和 Token 统计。
- [v0.10.0](v0.10.0.md)：Windows 安全凭据、代码审查和粘贴片段隔离。
- [v0.11.0](v0.11.0.md)：持续 Plan Mode、结构化计划与工作区过期检测。
- [v0.12.0](v0.12.0.md)：可恢复 Goal Mode、预算、证据与阻塞状态。
- [v0.13.0](v0.13.0.md)：工作区项目记忆、检索、来源与自动失效。
- [v0.14.0](v0.14.0.md)：计划范围执行、读取指纹、依赖上下文和完成验证。
- [v1.0.0](v1.0.0.md)：三平台凭据、Doctor、发布打包与稳定 CLI。

版本文档至少包含：版本目标、完成内容、使用方法、测试、已知限制、下一版本和建议提交信息。
