# CodeMuse v0.1.0

发布日期：2026-07-13

对应提交：`7545648 feat: initialize CodeMuse CLI`

## 版本目标

建立一个能够启动、持续输入、显示流式事件并支持多模型配置的 CLI 基线，为后续工具调用和代码库分析提供稳定入口。

## 已完成功能

- 交互式终端 REPL。
- 工作区、模型和运行模式展示。
- `/help`、`/model`、`/workspace`、`/clear`、`/cancel`、`/exit`。
- AgentRunner 与 AgentEvent 基础协议。
- MockAgent 本地演示。
- 兼容 DeepSeek、GLM 和自定义接口的流式模型调用。
- API Key 缺失时自动进入 Mock 模式。
- 5 项基础单元测试。

## 使用方法

```powershell
node src\cli.ts .
```

测试：

```powershell
node tests\run.ts
```

## 已知限制

- 不能读取或搜索项目代码。
- 不能修改文件或执行 Shell。
- ModelAgent 仅提供终端问答，不是真正工具调用 Agent。
