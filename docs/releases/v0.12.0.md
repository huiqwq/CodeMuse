# CodeMuse v0.12.0

发布日期：2026-07-17

## 版本目标

支持可在 CLI 重启后继续推进的长期目标，同时用预算、证据和停止条件限制自治范围。

## 完成内容

- `/goal create/status/pause/resume/complete/cancel/history`。
- 每个工作区最多一个未结束 Goal。
- 保存成功标准、子任务、Token、运行次数、累计时间、验证证据和最近失败。
- 成功验证可自动完成目标；预算耗尽或重复失败进入 `blocked`。
- Goal 仅在 CLI 前台推进，不启动守护进程。

## 使用方法

```text
/goal create 完成鉴权模块并通过测试
/goal resume
/goal status
/goal pause
```

## 测试与限制

自动测试覆盖唯一活动目标、预算累计、证据和完成状态。写入和执行继续遵循授权策略；退出进程不会在后台继续工作。
