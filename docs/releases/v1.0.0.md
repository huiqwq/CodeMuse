# CodeMuse v1.0.0

发布日期：2026-07-17

状态：代码与本地自动验收完成；npm 发布和三平台真实机器冒烟验收需由发布者执行。

## 版本目标

固化 Plan、Goal、Memory、准确执行和原有修改验证闭环，形成可通过 npm 全局安装的稳定 CLI。

## 完成内容

- npm 包版本 `1.0.0`、公开发布配置、文件白名单和 `prepack` 质量门。
- 首次启动配置提示和 `/doctor`、`/doctor export`。
- Windows DPAPI、macOS Keychain、Linux Secret Service 安全凭据后端。
- 工作区设置和脱敏诊断事件；不记录源码、完整 Diff、模型原文或密钥。
- Plan、Goal、Memory 和 Settings 独立版本化数据文件。
- 77 项自动测试、TypeScript 检查和 CLI 语法检查。
- 命令参考、隐私说明、迁移指南和排障文档。

## 安装和验收

```bash
npm install -g codemuse-cli
codemuse .
```

进入 CLI 后：

```text
/doctor
/model init
/plan on
/goal status
/memory list
```

## 发布前人工门槛

- 在 Windows、macOS、Linux 真实登录会话中分别验证 DPAPI、Keychain、Secret Service 往返。
- 验证全新全局安装、从 v0.10.0 升级和卸载。
- 使用至少一个真实 Provider 验证流式文本、Tool Calling、Token usage 和超时取消。
- 执行 `npm pack --dry-run`，确认包中没有 `.codemuse`、密钥或开发临时文件。

## 已知限制

- Goal 不在后台运行。
- 不提供任意 Shell、依赖自动安装或 Git 写操作。
- macOS Keychain 与 Linux Secret Service 需要有效桌面登录会话；无安全后端时不会回退明文存储。
- npm 注册表名称可用性和真实三平台凭据验收属于发布环境步骤，本地代码验收不能替代。
