# CodeMuse v0.14.0

发布日期：2026-07-17

## 版本目标

减少遗漏关联文件、陈旧补丁和无验证完成声明，并提供严格受计划约束的可配置自治。

## 完成内容

- 上下文选择增加 import、反向关联和测试文件关系加权。
- `read_file` 返回完整文件 SHA-256；`apply_patch` 写入前核对读取指纹。
- 已批准计划通过 `executionScope` 在 ToolRegistry 层限制写入路径。
- `apply_patch_set` 一次预览并原子应用 2—10 个已读取文件的协调变更。
- `/approval strict|plan-scoped`，默认 `strict`。
- `plan-scoped` 只自动授权计划内普通写入和计划列出的验证命令；删除、重命名继续逐项确认。
- ModelAgent 在完成前检查影响面、Diff 和验证证据；修改后没有成功验证会明确标为未验证。
- `complete` 事件增加 `verified` 和 `validationCommands`。

## 测试与限制

自动测试覆盖范围越界拒绝、读取后外部修改和授权设置持久化。符号关系当前针对常见 JavaScript/TypeScript import 语法，不替代语言服务器的完整语义分析。

## 下一版本

v1.0.0 完成跨平台凭据、诊断、打包和正式发布文档。
