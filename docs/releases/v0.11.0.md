# CodeMuse v0.11.0

发布日期：2026-07-17

## 版本目标

把原有的固定任务步骤升级为持续式 Plan Mode。规划期间只允许扫描、读取和搜索，计划批准前不能写文件或执行项目命令。

## 完成内容

- `/plan on/status/revise/approve/off` 完整状态流。
- `PlanArtifact` 保存目标、范围、步骤、验证、风险、假设和修订记录。
- `.codemuse/plan.json` 版本化、限长、原子写入和未知字段校验。
- 计划保存工作区 SHA-256 检查点；漂移后标记 `stale`。
- 批准后把计划范围传入 ToolRegistry，范围外写入由执行入口拒绝。

## 使用方法

```text
/plan on
实现登录重试
/plan revise 只修改网络层，不调整 UI
/plan approve
```

## 测试与限制

自动测试覆盖创建、修订、持久化和漂移检测。计划步骤由模型输出和实际上下文归纳，仍必须以执行时读取到的代码为准；计划不会自动扩大写入范围。
