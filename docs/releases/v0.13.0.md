# CodeMuse v0.13.0

发布日期：2026-07-17

## 版本目标

为重复任务提供可审计、可失效的项目级长期记忆，不跨项目共享内容。

## 完成内容

- `/memory list/show/add/forget/clear`。
- 记忆类型、来源、关联路径、置信度、验证时间和失效键。
- 关联文件使用 SHA-256 指纹；文件变化后记忆自动标记 `stale`。
- 检索使用独立 1000 Token 预算，最多向任务提供 8 条有效记忆。
- 密钥、Bearer Token 和疑似 API Key 内容拒绝保存。
- `.codemuse/memory.json` 版本化、原子写入并限制为 200 条。

## 使用方法

```text
/memory add src/api/client.ts 的重试策略必须保持幂等
/memory list
/memory forget 12ab34cd
```

## 测试与限制

自动测试覆盖工作区隔离、敏感信息拒绝、检索和文件变化失效。模型推测不会自动写成长期事实；自动注入的记忆始终标注为需要重新验证的线索。
